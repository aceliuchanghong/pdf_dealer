### UI

```shell
python entity_extract_ui.py
nohup python entity_extract_ui.py>entity_extract_ui.log &
```

### Env

```shell
vi .env
conda create -n pdf_dealer python=3.11 -y
conda activate pdf_dealer
pip install -r requirements.txt
```
